# ncaaSchedulesAPI

## Description
 - Ncaa Schedules API aims to gather and provided college sports schedules for developers to use
 - Getting accurate and always up to date 
 - beta version (football only)
 
## Technologies
 - CHEERIO JS
 - NODE JS
 - REST API
 - MONGO DB
